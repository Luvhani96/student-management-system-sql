# 🎓 Student Management System (MySQL)

A professional relational database project built using MySQL to manage students, departments, lecturers, courses, enrollments, marks, attendance, and fees.

## 🚀 Features

* Student record management
* Course registration
* Marks and performance tracking
* Attendance management
* Fee payment records
* SQL joins and reports
* Views, Procedures, Triggers

## 🛠️ Tech Stack

* MySQL
* MySQL Workbench
